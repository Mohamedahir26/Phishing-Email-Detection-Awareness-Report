# Phishing Email Detection & Awareness Report

## 1. Project Overview
This project involves a detailed analysis of a phishing email campaign impersonating Netflix. The goal is to identify malicious indicators, classify the threat, and provide professional remediation strategies.

## 2. Key Features Included
- **Identification:** Analysis of spoofed sender addresses and hidden URLs.
- **Classification:** Categorized as a High-Risk Phishing attempt.
- **Techniques:** Explanation of social engineering tactics used by attackers.
- **Guidelines:** Prevention steps for employees and organizations.

## 3. Tools Used
- **Email Header Analyzer**.
- **Browser Developer Tools**.
- **Microsoft Word / PDF**.
